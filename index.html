Responsive web design is inarguably a great way to build futureproof websites and applications. But the one thing CSS media queries can't always help us with is optimizing our users' interactions depending on their output device.

Responsive JavaScript—finally

Harvey executes certain parts of your JavaScript based on a media query condition. It gives you complete control over all changes through three easy-to-use callbacks:

setup
Called the first time the media query becomes valid. Use the setup callback to load extra assets or manipulate the DOM.
on
Called every time the media query becomes valid. Use the on callback to reveal changes you made (e.g. in setup).
off
Called every time the media query becomes invalid. Use the off callback to revert/hide changes you made (e.g. in setup).
Harvey.attach( 'screen and (min-width: 40em)', {
  setup: function() {},
  on: function() {},
  off: function() {}
});
You can repeat this step multiple times and for any valid media query you need.
